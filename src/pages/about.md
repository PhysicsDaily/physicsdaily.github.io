# About PhysicsDaily

Welcome to **PhysicsDaily**, a comprehensive open-source resource for university-level physics notes, simulations, and practice problems.

## Our Mission
Our goal is to make high-quality physics education accessible and interactive. We believe that learning physics should involve more than just reading text; it should involve playing with variables, visualizing concepts, and testing your understanding.

## The Tech Stack
This website is built with:
- **Docusaurus**: For the documentation and site structure.
- **React**: For interactive components and simulations.
- **KaTeX**: For beautiful mathematical rendering.

## Contributing
This project is open source! If you spot an error or want to add a new simulation, feel free to contribute on [GitHub](https://github.com/PhysicsDaily/physicsdaily.github.io).

## Contact
Have questions or suggestions? Open an issue on our GitHub repository.
